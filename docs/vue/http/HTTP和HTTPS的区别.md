## 一、HTTP

`HTTP（HyperText Transfer Protocol）`，
即超文本运输协议，
是实现网络通信的一种规范。

![HTTP 的解释](../images/http/HTTP和HTTPS的区别/1.png)

在计算机和网络世界有，
存在不同的协议，
如：
`广播`协议、
`寻址`协议、
`路由协议`等等。。。

然而`HTTP`是一个传输协议，
即将数据由`A`传到`B`
或`B`传输到`A`,

并且`A`与`B`之间能够存放很多第三方，

如：`A<=>X<=>Y<=>Z<=>B`。

传输的数据并不是计算机底层中的二进制包，
而是完整的、
有意义的数据，
如`HTML`文件，
图片文件，
查询结果等超文本，
能够被上层应用识别。
